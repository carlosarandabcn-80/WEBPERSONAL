# Carlos Aranda Sanchez - Portfolio Web

Landing page profesional lista para publicar en GitHub Pages.

## Que incluye

- `index.html`: estructura completa de la web.
- `assets/css/styles.css`: diseno responsive, modo claro/oscuro, glassmorphism y microinteracciones.
- `assets/js/main.js`: navegacion mobile, animaciones on-scroll, filtros de portfolio, asistente demostrativo y canvas dinamico.
- `assets/images/carlos-cover-hero.webp`: portada editorial optimizada en escala de grises.
- `assets/images/carlos-cover-hero.png`: fallback PNG de la portada.
- `assets/images/carlos-studio-portrait.webp`: retrato secundario de apoyo visual.
- `assets/images/favicon.svg`: favicon de marca.

## Publicacion en GitHub Pages

1. Sube esta carpeta a un repositorio de GitHub.
2. En GitHub, entra en `Settings > Pages`.
3. Selecciona la rama principal y la carpeta raiz del repositorio.
4. Guarda los cambios. GitHub generara la URL publica.

## Stack propuesto

Esta entrega es estatica para facilitar GitHub Pages. Si se quiere evolucionar a producto, el siguiente paso natural seria:

- Next.js o Astro para componentes y SEO avanzado.
- Tailwind CSS si se desea un sistema de diseno escalable.
- Framer Motion o GSAP para animacion avanzada.
- Three.js para una escena 3D real si se anade identidad visual interactiva.
- Un asistente AI conectado a una base documental del CV, proyectos y disponibilidad.

## SEO y accesibilidad

- HTML semantico con secciones identificables.
- Meta title, description, Open Graph y favicon.
- Navegacion por anclas y `skip link`.
- Respeto a `prefers-reduced-motion`.
- Contraste adaptado para modo claro y oscuro.
